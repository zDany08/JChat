# JChat
### How to run Client
Execute jar file by double-clicking on it or run it by command prompt with the following code:
> java -jar Client.jar
### How to run Server
Run jar file by command prompt with the following code:
> java -jar Server.jar 25565

25565 is default port, you can change it to the one you need.
